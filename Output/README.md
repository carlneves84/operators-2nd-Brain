# Output

Skill outputs and analysis documents. Anything produced in a session that is worth keeping but is not a decision log entry, a handoff, or a weekly note goes here.

Examples of what belongs here:
- Experiment briefs produced by `grill-me`
- Red-team reports from `contrarian-advisor`
- Diagnostic write-ups from `diagnose`
- Draft copy and analyses
- One-pagers and strategy docs

No strict naming convention required, but kebab-case dates help: `2026-05-15-activation-audit.md`.
