# Weekly

Weekly planning notes, one per week. Named `YYYY-MM-DD.md` where the date is the Monday of that week.

The `weekly-planner` skill generates these automatically, pulling from your documentation platform, meeting notes, and last week's note.

## Structure

Each note follows this format:

```markdown
# Week of [Mon DD] - [Fri DD], YYYY

## Weekly Goals
- [ ] [top priority for the week]

## Carryover from Last Week
- [ ] [incomplete tasks from the previous note]

## Meeting Actions
### [Meeting Name]
- [ ] [action item]

## Follow-ups
- [ ] [ongoing items to monitor]

## Experiments to Monitor
- [Experiment name] ([status])

## Daily Log
### Monday YYYY-MM-DD
### Tuesday YYYY-MM-DD
...

## Notes
### Key Decisions from This Week
### Wins from This Week
### Carry-Forward Watch List
```

If you are not using `weekly-planner` yet, copy this template and fill it in manually on Friday afternoons.
